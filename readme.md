## Product Overview

**Name:** Sendry  
**Tagline:** Protect your email reputation. Reach the inbox.  
**Mission:** Reduce email deliverability failures by providing early visibility into sender reputation, authentication issues, and risk signals before they impact campaigns.

---

## Current Status

- PRD and system design completed  
- Tech stack finalized  
- Core services under active development  
- Not production-ready  

---

## Problem

Email senders often face:
- Inbox placement drops without clear reasons  
- Misconfigured SPF, DKIM, or DMARC  
- Poor visibility into domain/IP reputation  
- Expensive or overly complex enterprise tools  

---

## Solution

Sendry provides a lightweight, developer-friendly platform to:
- Monitor deliverability health continuously  
- Detect configuration and reputation issues early  
- Receive actionable alerts before damage occurs  

---

## Key Features (Planned)

- Domain & IP reputation monitoring  
- SPF, DKIM, DMARC validation  
- Deliverability health score  
- Bounce and spam complaint tracking  
- Blacklist detection  
- Email alerts for risk events  
- Multi-domain support  

---


## Tech Stack

- **Frontend:** Next.js + TypeScript + Tailwind CSS  
- **Backend:** Node.js (Express / NestJS)  
- **Database:** PostgreSQL  
- **Caching & Queues:** Redis  
- **Auth:** JWT  
- **Infra:** Docker + Cloud (AWS / GCP / Vercel)
